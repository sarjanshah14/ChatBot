# ChatBot Projects

This repository contains two chatbot implementations:

1.  **app.py**: A chatbot powered by the Gemini API.
2.  **app_backup.py**: A chatbot utilizing Retrieval-Augmented Generation (RAG).

## Usage

To run the main Gemini chatbot:
```bash
python app.py
```

To run the RAG chatbot:
```bash
python app_backup.py
```
